Pre-req software:

1) .Net Framework 4.6.1
2) System.Data.Sqlite (Included in this folder, see "Install Steps" for more information). 
  At this time, I'm not sure if this is needed for web servers or not.

Install Steps:

1) Run sqlite-netFx46-setup-bundle-x86-2015-1.0.106.0
   1.a) Check all options
2) Run sqlite-netFx45-setup-bundle-x64-2012-1.0.106.0
   2.a) Take defaults
3) Import RepopCompanion_TemplatePage into Visual Studio
